# RealWorld Flutter App

This is a [RealWorld](https://github.com/gothinkster/realworld) Flutter app.

# Quick Installation

```dart
git clone git@github.com:rhalff/realworld_flutter.git
flutter pub get
flutter pub run build_runner build
flutter run 
```
